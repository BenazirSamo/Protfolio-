About Me

Get to know me better

Frontend Developer with Passion for Creating Interactive Web Experiences

Hello! I’m Benazir, a passionate Frontend Developer based in Karachi, Pakistan. I’m currently pursuing a diploma in AI Software Engineering from Aptech Metro Star Gate and constantly working to expand my skills to become a full-stack web developer.

I specialize in building responsive and visually appealing websites using HTML, CSS, JavaScript, and Bootstrap. I’m also experienced with tools like GitHub for version control and Canva for creating clean, modern UI designs.

My journey began with a strong curiosity for technology, which I’ve turned into hands-on skills through consistent learning and practice. I’ve created several real-world projects, including a jewelry website,Vinegar-Food-Restaurant,coffee shop website, and various mini web apps like digital clocks, to-do lists, notes tools, trivia quizzes, calculator apps, and more.

I’m detail-oriented, self-driven, and always excited to learn new technologies. My goal is to build meaningful, user-friendly digital experiences that combine design with functionality.
